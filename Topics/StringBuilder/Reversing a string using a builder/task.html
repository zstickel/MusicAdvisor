Reversing a string using a builder

Hey there! This problem might be a bit unpredictable, but give it a go and let us know how you do!
Write a program that reads a string as input and reverses it using StringBuilder. The input string will contain only lowercase English letters and will have a maximum length of 100 characters. Print the reversed string as output.
Sample Input:
hello

Sample Output:
olleh


Sample Input:
racecar

Sample Output:
racecar



Memory limit: 256 MB
Time limit: 5 seconds
